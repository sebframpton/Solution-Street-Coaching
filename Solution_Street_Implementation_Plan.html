Solution Street
Coaching Services
Website Implementation Plan

Prepared for Jason Sammut — Founder & CEO

Date: 4 March 2026

Version 1.0

1. Overview

Build a modern, premium single-page website for Solution Street Coaching Services, an NDIS disability support provider founded by Jason Sammut. The site will communicate the company's mission, values, and full list of services, while providing a clear path for potential clients to get in touch.

2. Technology Stack

Layer Choice Rationale
Scaffold Vite (vanilla JS) Fast dev server, zero-config, no framework overhead
Structure HTML5 Semantic, accessible, SEO-friendly
Styling Vanilla CSS Full control, no dependency
Interactivity Plain JavaScript Lightweight, no library needed
Fonts Google Fonts (Outfit + Inter) Modern, accessible typography

3. Design System

Colour Palette

Token Value Usage
Navy #0a1628 Backgrounds, headings
Teal #1abc9c Primary accent, buttons, links
Teal Dark #16a085 Hover states, gradients
Warm White #f5f5f5 Text on dark, light backgrounds
Soft Gold #f0c27f Highlight accents

Typography

  • Headings: Outfit (600–700 weight)
  • Body: Inter (400–500 weight)

Visual Style

  • Dark-mode-first aesthetic
  • Glassmorphism cards with backdrop blur
  • Smooth gradients and subtle noise texture
  • Responsive: mobile-first with breakpoints at 768 px and 1024 px

4. Page Sections

The website is a single-page design with smooth-scrolling anchor links between sections.

Section Content
Navbar Sticky nav with logo text, anchor links (About · Services · Contact), mobile hamburger menu
Hero Full-viewport section with gradient overlay, headline "Empowering Lives Through the NDIS", Jason's intro copy, CTA button
About Two-column layout — founder quote/story on the left, values (Communication, Hard Work, Results) as animated icon cards on the right
Services 11 service cards in a responsive CSS grid with hover-lift and glow effects (see full list below)
CTA Banner Full-width gradient strip — "Don't look any further — join Solution Street" with CTA button
Contact Contact form (Name, Email, Phone, Message) + address / phone / email details
Footer Logo, quick links, copyright © 2026, social media icon placeholders

6. Interactive Features

  • Smooth scroll for anchor navigation
  • Intersection Observer scroll-reveal animations on sections and cards
  • Mobile hamburger navigation (slide-in menu)
  • Service card hover effects (lift + glow)
  • Lightweight contact form validation with user-friendly error states

7. SEO & Accessibility

  • Descriptive <title> and <meta description>
  • Open Graph tags for social media sharing
  • Single <h1> per page with proper heading hierarchy
  • Semantic HTML5 elements (<nav>, <main>, <section>, <footer>)
  • Alt text on all images
  • Keyboard-navigable interactive elements

9. Items for Review

⚠ Brand Colours & Logo The proposed palette is dark navy + teal/green. If Solution Street has existing brand colours or a logo file, these will be incorporated into the design instead.
⚠ Contact Form The contact form will be front-end only (no emails sent) for the initial build. A service like Formspree or Netlify Forms can be added later for actual email delivery.

10. Verification Plan

Browser Testing

  • Run local dev server and open in browser
  • Verify navbar sticky behaviour and mobile hamburger toggle
  • Verify hero section with background and CTA
  • Verify about section layout and animated cards
  • Verify all 11 service cards display with hover effects
  • Verify contact form with validation feedback
  • Verify footer with links
  • Check responsive layout at mobile width (≤ 768 px)

Manual Verification

  • Client review in their own browser
  • Confirm design, content, and responsiveness
